# Global Cybersecurity Threats Analysis (2015–2024) – Power BI Dashboard

## Project Overview
This project presents an interactive **Power BI dashboard** built using the **Global Cybersecurity Threats (2015–2024)** dataset from Kaggle.  
The dashboard analyzes global cybersecurity incidents by exploring patterns in **attack types, target industries, financial losses, affected users, attack sources, vulnerabilities, defense mechanisms, and incident resolution times**.

The objective of this project is to visualize cybersecurity trends and provide insights into the **impact and evolution of cyber threats across countries and industries**.

---

## Project Objectives

- Analyze global cybersecurity threat patterns  
- Identify the most affected industries and countries  
- Evaluate financial impacts of cyberattacks  
- Visualize cybersecurity data using Power BI dashboards  

---

## Dataset Information
  
- **Source:** Kaggle  
- **Type:** Cybersecurity Incident Dataset  
- **Total Columns:** 10  

### Dataset Features

| Column | Description |
|------|-------------|
| Country | Country where the cyber incident occurred |
| Year | Year of the cybersecurity incident |
| Attack Type | Type of cyberattack (e.g., phishing, malware) |
| Target Industry | Industry targeted by the attack |
| Financial Loss (in Million $) | Estimated financial loss caused by the attack |
| Number of Affected Users | Number of users impacted |
| Attack Source | Origin/source of the cyberattack |
| Security Vulnerability Type | Vulnerability exploited in the attack |
| Defense Mechanism Used | Security method used to mitigate the attack |
| Incident Resolution Time (in Hours) | Time taken to resolve the cybersecurity incident |

---

## Dashboard Features

The Power BI dashboard provides insights such as:

- Cyberattack trends from **2015–2024**
- Most frequent **attack types**
- **Industries most targeted** by cyberattacks
- **Financial loss analysis** caused by attacks
- Countries with the **highest number of affected users**
- Analysis of **attack sources and security vulnerabilities**
- **Defense mechanisms** used against attacks
- Comparison of **incident resolution times**
